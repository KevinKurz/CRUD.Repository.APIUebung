# Reservation-Manager BackEnd

My attempt to create a virtual table reservations system from my first year as a trainee.
You can send a reservation request to the API with the number of poeple you want to visit, your range of time, date and your name.
It will automaticly place your reservation to one of the suited tables and will give you feedback, weather your reservation is accepted or declined.

### Note: 
This Backend only runs local.
Currently this will generate a file in the /bin folder of the project, to store the tables and reservations as a JSON-file.
